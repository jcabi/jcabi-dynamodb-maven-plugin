jcabi-dynamodb:stop

Full name:

com.jcabi:jcabi-dynamodb-maven-plugin:0.10.0:stop

Description:

Stops DynamoDB Local.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 0.1.
  • Binds by default to the lifecycle phase: post-integration-test.

<arguments> List 0.5 Command line arguments of DynamoDBLocal.

Default value is: 10101.

Default value is: false.
